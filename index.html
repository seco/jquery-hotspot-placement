jQuery Hotspot Placement

This is a tool for placing hotspots on an image, meant to be used on the backend.

This is useful for catalogs, where you can place a hotspot on top of a product in the image (say in a kitchen or living room) and when the hotspot is clicked a modal will be displayed. Placing each hotspot manually is a pain, especially if there are more than a few catalog images and more than 5 products.

The style of this interface is based on the Django admin. It uses jQuery and the Draggable widget from jQuery UI.
